# Frontend - Level 2

Things that you will have to learn to pass the level 2:
- History of the MVC frameworks, and modern frameworks
  - Understand the story from vanilla JS, to jquery, backbone, to react.
- Understand the concept of an application router
- Understand the separation of concepts
  - Views
  - Services
  - Data models
- Develop an application with jQuery
- Create your first API
